# Linter Feature for Aura Text

## Overview
The integrated linter provides linting functionality for Python files.

## Features
- **Powerful PyLint backend:** The linting functionality uses the very powerful tool called PyLint, which is one of the most powerful Python linters.
- **Manual & Opt-in, not Opt-out:** The linter is fully manual (needs to be triggered from menu bar), and so it does not run unless you run it yourself.
- **Simple & easy to use:** The linter is easy and intuitive to use, and a docked widget automatically appears with all errors and warnings from the linter.

## Customization
You can customize linter behavior by creating a config file for pylint at `.pylintrc`, which can control the CPU usage of it.
